# StellarMarket Smart Contracts

This directory contains smart contracts for various blockchain platforms:

- `stellar/` - Stellar Network smart contracts (SCPs, Soroban contracts)
- `ethereum/` - Ethereum Virtual Machine (EVM) compatible contracts
- `solana/` - Solana Program Library (SPL) contracts

## Development

Each subdirectory contains its own development setup and documentation.
