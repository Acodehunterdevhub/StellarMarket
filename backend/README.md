# StellarMarket Backend

API server and business logic for the StellarMarket decentralized marketplace.

## Getting Started

### Prerequisites
- Node.js >= 16.0.0
- npm >= 8.0.0

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

## Scripts

- `npm run dev`: Start development server with nodemon
- `npm run build`: Compile TypeScript to JavaScript
- `npm run start`: Start production server
- `npm run test`: Run tests
