# StellarMarket Frontend

Next.js application for the StellarMarket decentralized marketplace.

## Getting Started

### Prerequisites
- Node.js >= 16.0.0
- npm >= 8.0.0

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

## Scripts

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run start`: Start production server
- `npm run lint`: Run ESLint
- `npm run test`: Run tests
